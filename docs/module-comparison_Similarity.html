(static) discretizeTime(track, secondsPerBin)

Source:
  • comparison/Similarity.js, line 130
- Normalizes Note times to be between 0 and (maxTime - minTime), - discretizes the start and end time by using Math.round to get the closest time bin (beat) and - Creates one array for each pitch, where each entry contains either a 0 (no note at that time bin) or a 1 (note at that time bin)
Parameters:
Name Type Description
track Array.<Note> an array of Note objects
secondsPerBin number time bin size in seconds

(static) getSimilarParts(track, selectedInterval, stride, threshold, secondsPerBin, distance)

Source:
  • comparison/Similarity.js, line 22
Given a track, a selected time interval and a threshold, this function searches for parts in the track that are similar to the selection. It uses a sliding window with the size of the selection and a stride given as argument.
Parameters:
Name Type Description
track Array.<Note> array of Note objects
selectedInterval Array.<number> [startTime, endTime] in seconds
stride number stride for the sliding window in number of bins
threshold number distance threshold below which parts are considered similar
secondsPerBin number time bin size in seconds
distance string one of: 'dtw', 'euclidean', 'nearest'

(static) getTrackSimilarity(discrA, discrB, distance)

Source:
  • comparison/Similarity.js, line 78
Uses dynamic time warping (DTW) to calculate the distance between two discretized tracks, for each pitch separately. Pitch-wise distances are averaged and a penalty if added to the distance for pithces that are not occuring in both tracks https://github.com/GordonLesti/dynamic-time-warping
Parameters:
Name Type Description
discrA Map discretized track
discrB Map discretized track
distance string one of: 'dtw', 'euclidean', 'nearest'

(inner) neirestNeighborDistance(A, B)

Source:
  • comparison/Similarity.js, line 214
Given two arrays containing 1s and 0s, this algorithm goes through all bins and for each bin where one array has a 1 and the other a 0, it searches for the closest 1 next to the 0. The distance is then added to the global distance.
